Detailed Description

This file is the sd card driver using SPI It has been implemented, using the SD/MMC using SPI Example for EVK1100 Documentation from ATMEL (http://asf.atmel.com/docs/2.9.0/avr32.components.memory.sdmmc.spi.example.evk1100/html/index.html)
