Detailed Description

System Control InterFace(SCIF) driver.

  • Compiler: IAR EWAVR32 and GNU GCC for AVR32
  • Supported devices: All AVR32 UC3C devices.
  • AppNote:
Author
Atmel Corporation: http://www.atmel.com
