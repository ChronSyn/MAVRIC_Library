Detailed Description

TWI Master driver for AVR UC3.

This file defines a useful set of functions for the TWIM interface on AVR UC3 devices.

  • Compiler: IAR EWAVR32 and GNU GCC for AVR UC3
  • Supported devices: All AVR UC3 devices with an TWIM module can be used.
